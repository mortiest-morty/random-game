# random-game
Компьютер загадывает случайное число в заданномдиапазоне, а игрок должен угадать его. После каждой попытки программа сообщает, больше или меньше загаданного числа предложенное игроком значение.

# Как играть
1. Запустите скрипт
2. Введите цифры при появлении запроса
3. Получайте подсказки, если ваш номер больше или меньше
4. Выигрывайте, если угадаете правильный номер!
